Wally Weather

About the Data:

Introduction

Using this scraper by Max Fan, we were able to collect 51K+ days worth of hourly weather data. This data was pulled from api.weather.com. We collected data from airports in New Haven, New York City, and Boston.

Dataset Contents:

Currently, the dataset only contains 1 GB of data from three separate weather stations.
